## Tools to need
* MainBox
 * 1x Raspberry PI3
 * 1x RFM69 Module
 * 1x 10.1" HDMI Touch LCD

* PlayBox
 * 1x Arduino Mega
 * 1x RFM69
 * 25x Neopixels WS2812S 5050 RGB LED
 * 5x Pushbutton
 * 1x Speaker Piezo


## Connect the Tools

Attach the RFM69 as follows:
RFM pin - Pi pin
3v3 - 17
DIO0 - 18 (GPIO24)
MOSI - 19
MISO - 21
CLK - 23
NSS - 24
Ground - 25
RESET - 29
